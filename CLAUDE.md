# 楽天トレンドワード収集システム - CLAUDE.md

## プロジェクト概要

楽天市場のカテゴリ別ランキングから毎日トレンドワードを自動収集・分析するシステム。
中国輸入ビジネスの商品リサーチ用途。Google Apps Script (GAS) で動作し、深夜2時に自動実行。

## Google Sheetsの構成

**スプレッドシートID**: 

| シート名 | 内容 |
|---|---|
| 設定 | ジャンル名・楽天ランキングURL・有効フラグ・KeepaカテゴリID |
| ワード集計 | 日付・ジャンル・キーワード・出現回数・平均順位・スコア・分類・新規参入フラグ |
| 商品一覧 | 日付・ジャンル・キーワード・分類・順位・商品名・商品URL |
| 除外ワード | 有効(TRUE/FALSE)・ワード・種類・メモ ← チェックボックスで管理 |
| 除外候補 | 同上フォーマット ← 自動検出ワードのレビュー用 |
| サマリー | AI分析結果（手動実行時のみ） |

## スコアリング仕様

- **順位スコア**: 1位=300点、300位=1点（線形）
- **分類**:
  - （🎯隠れた狙い目）: 出現1〜3回、スコア×1.5倍ブースト
  - （📈注目ワード）: 出現4〜9回
  - （⚠️飽和状態）: 出現10回以上
- **新規参入フラグ**: 過去30日間に登場していないキーワードに付与

## 除外ワードシートの列構造

**重要**: 列順が特殊

- A列: 有効（TRUE/FALSE チェックボックス） → 
- B列: ワード → 
- C列: 種類（\!マーケ / \!品質 / デザイン / 機能 / 季節タグ / 状態 / 汎用語 / 補助語）
- D列: メモ

TRUE のワードのみ除外される。種類名の  はソート用プレフィックス（意味なし）。

## 設定シートの列構造

- A列: ジャンル名
- B列: 楽天ランキングURL（例: https://ranking.rakuten.co.jp/daily/100804/）
- C列: 有効（チェックボックス TRUE/FALSE）
- D列: KeepaカテゴリID（任意）
